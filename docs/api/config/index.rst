``configuration.py`` API
========================

The ``configuration.py`` file is the standard way to configure wildcat settings. These pages document the settings available from the config file.

----

:doc:`IO Folders <folders>`
    Settings that determine the folders where wildcat will look for input files and save output files.
    
:doc:`preprocess <preprocess>`
    Settings used to run the preprocessor.

:doc:`assess <assess>`
    Settings used to run an assessment.

:doc:`export <export>`
    Settings used to export assessment results.
