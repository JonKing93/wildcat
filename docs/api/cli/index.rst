Command Line API
================

These pages provide a complete reference guide to using the command line interface (CLI).

----

:doc:`wildcat initialize <initialize>`
    Initialize a project from the command line.

:doc:`wildcat preprocess <preprocess>`
    Preprocess input datasets from the command line.

:doc:`wildcat assess <assess>`
    Conduct a hazard assessment from the command line.

:doc:`wildcat export <export>`
    Export saved assessment results from the command line.
