Command Overviews
=================

These pages provide overviews of the main wildcat commands. They explore the key steps of each command and describe the most commonly configured settings. 

----

:doc:`initialize`
    Creates a project folder for an assessment, including a default configuration file.

:doc:`preprocess`
    Reprojects and cleans input datasets in preparation for an assessment.

:doc:`assess`
    Conducts a hazard assessment using preprocessed datasets.

:doc:`export`
    Exports hazard assessment results to common GIS formats (such as Shapefiles and GeoJSON)
