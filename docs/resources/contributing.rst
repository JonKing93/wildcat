Contributing
============

We welcome feedback and contributions, such as: suggestions for new features, bug reports, and code contributions. To share feedback, you can:

* `Create an Issue <https://code.usgs.gov/ghsc/lhp/pfdf/-/issues/>`_
* Contact the developers, or
* Contact the `Landslides Hazards Program <https://www.usgs.gov/programs/landslide-hazards>`_

If you plan to contribute code, then please read the :doc:`Developer Guide <dev-guide>`.
